# engine_reference — ЭТАЛОН для сверки (не движок приложения)

Это уже работающий, проверенный код на Swiss Ephemeris (`pyswisseph`/`pysweph`).
Назначение — **эталон**: новый движок приложения (WASM или Chaquopy) обязан
давать идентичные результаты. Сверка — до 0.01° по долготам и до минуты по
временам точных аспектов. Подробности — в `../ЗАДАНИЕ.md`, раздел 9.

Файлы:

- `daily_aspects_astro.py` — мажорные аспекты на день: точное время кульминации,
  applying/separating (→/←), аудит элонгаций, список объектов `BODIES`,
  словарь аспектов `ASPECTS`, глифы. Главная функция `aspects_for_day()`.
- `daily_summary.py` — форматирование дневной сводки (образец состава полей
  карточки аспекта).
- `transits_astro.py` — аспекты/соединения в диапазоне дат, **окно орбиса**
  (вход → точный момент → выход) с обработкой ретроградного «качания»
  (`_orb_window`, `transit_conjunctions`); плюс натальная карта и горизонтальная
  система домов — на будущее.
- `transits_files.py` — генерация `.ics` (календарь) и `.txt` — образец, если
  понадобится экспорт событий.

Запуск эталона требует окружения Python 3.12 с `pysweph` (см. `../CLAUDE.md`).
